Social Media

All stats have been derived from data extracted from Sprout Social. The raw data extracted from sprout is broken down by day and has been aggregated appropriately (either by summing up values or finding the latest value in a given data range). More information about the extracted data can be found in this article from Sprout. The data can be downloaded from Data Mill North.
